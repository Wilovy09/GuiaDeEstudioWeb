HTML

¿Qué es HTML?

Que va a ir y en donde va ir

HTML, o HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar utilizado para crear y diseñar páginas web. Es un lenguaje de marcado que permite estructurar el contenido de una página web mediante el uso de etiquetas o elementos que describen la semántica y la estructura del documento.
En términos más simples, HTML proporciona la estructura básica de una página web, definiendo los diferentes elementos que componen la página, como encabezados, párrafos, listas, enlaces, imágenes, formularios, entre otros. Estos elementos se organizan mediante etiquetas HTML, que son instrucciones delimitadas por corchetes angulares
("<" y ">") que indican al navegador web cómo mostrar el contenido. HTML es fundamental para la creación de páginas web ya que proporciona la base sobre la cual se construye el contenido y la apariencia visual de un sitio web.
Trabaja en conjunto con otros lenguajes como CSS (Cascading Style Sheets) para el diseño y la presentación visual, y JavaScript para la interactividad y la funcionalidad dinámica. Juntos, estos tres lenguajes forman la tríada básica de desarrollo web.

Estructura básica de HTML

	</html>

- Imágenes

Representadas por la etiqueta <img>, se utilizan para incrustar imágenes en una página web. Se debe proporcionar la URL de la imagen en el atributo "src".

Código Resultado
<img src="..."/>
Atributo Descripción
src Este atributo es usado para definir la ruta de la imágen que usaremos, puede ser una ruta de la web o una ruta relativa
alt Este atributo es usado para definir un texto alterativo por si la imágen no llega cargar o ser encontrada.
weight Este atributo es usado para definir la anchura de la imagen a utlizar, por defecto usa pixeles como medida.
height Este atributo es usado para definir la altura de la imagen a utlizar, por defecto usa pixeles como medida.

- Hipervínculos

Representados por la etiqueta <a>, se usan para crear enlaces a otras páginas web o recursos. Se debe proporcionar la URL de destino en el atributo "href".

Código Resultado
<a href="...">Link</a> Bootstrap

- Listas

Existen 3 tipos de listas.

Listas ordenadas

Representadas por la etiqueta <ol>, se utilizan para crear listas ordenadas. Cada elemento de la lista se define mediante la etiqueta <li>. 

					
	<ol>
		<li>Elemento 1</li>
		<li>Elemento 2</li>
		<li>Elemento 3</li>
	</ol>
  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Listas desordenadas

Representadas por la etiqueta <ul>, se utilizan para crear listas desordenadas. Cada elemento de la lista se define mediante la etiqueta <li>. 

					
	<ul>
		<li>Elemento 1</li>
		<li>Elemento 2</li>
		<li>Elemento 3</li>
	</ul>
  • Elemento 1
  • Elemento 2
  • Elemento 3

Listas de definición

Representadas por la etiqueta <dl>, se utilizan para crear listas de definición. Cada término de la lista se define mediante la etiqueta <dt>, y su definición se define mediante la etiqueta <dd>. 

					
	<dl>
		<dt>Termino 1</dt>
		<dd>Definición 1</dd>
		<dt>Termino 2</dt>
		<dd>Definición 2</dd>
	</dl>

CSS

¿Qué es CSS?

Como se ven las cosas.

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para definir la presentación y el aspecto visual de un documento HTML. Con CSS, puedes controlar la apariencia de elementos HTML, como el color, el tamaño, la fuente y el diseño. CSS permite separar la estructura del contenido HTML de su estilo, lo que facilita la creación de sitios web con un diseño coherente y flexible. Además, al utilizar reglas de estilo, CSS permite aplicar estilos de manera eficiente y consistente en todo un sitio web.

- Propiedades y valores

Las propiedades son características que deseas cambiar, como el color o el tamaño del texto. Los valores son los ajustes específicos que le das a esas propiedades. 

						
	p {
		/* Cambia el color del texto a azul */
		color: blue; 
		
		/* Tamaño de la letra en 16 píxeles */
		font-size: 16px;
	}

- Selectores

Por elemento
								
	p {
		color: red;
	}
Por clase
							
	.clase {
		color: red;
	}
Por ID
							
	#id {
		color: red;
	}
Selecciona todo (*)
							
	* {
		/* Las reglas que escribamos con este selector 
		se aplicaran a todos los elementos, 
		si es que estos no tiene reglas ya definidas */
	}
Combinación de clases
							
	.clase1.clase2 {
		/* Estilos para elementos que tienen 
		ambas clases "clase1" y "clase2" */
	}
Elemento con clase
							
	p.clase {
		/* Estilos para elementos p que también tienen 
		la clase "clase" */
	}
Selectores multiples
							
	p, .clase, #id {
		/* Estilos aplicados a los elementos p, .clase y #id */
	}

Hay tres formas de usar CSS en un documento HTML: en línea, incrustado y externo.

En línea

Este es el resultado con los estilos en línea.

- Variables

Las variables en CSS se crean usando la siguiente sintaxis:

					
	/* Aqui agregaremos todas nuestras vaiables CSS */
	:root{
		--color-Cafe: brown;
	}

Para hacer referencia a la variable se utiliza la funcion var().

					
	/* Aqui usamos la variable que creamos anteriormente */
	p {
		color: var(--color-Cafe);
	}

- El modelo de caja

El modelo de caja es un concepto fundamental en CSS que describe cómo se representan los elementos HTML en una página web. Cada elemento HTML se representa como una caja rectangular que consta de cuatro áreas principales:

Margen / Margin
Es el espacio entre el borde de la caja y los elementos adyacentes. Los márgenes ayudan a controlar el espacio entre los diferentes elementos en la página.
Borde / Border
Es la línea que rodea el contenido y el relleno de la caja. Puede tener un grosor, un estilo (como sólido, punteado, etc.) y un color definidos.
Relleno / Padding
Es el espacio entre el contenido y el borde de la caja. El relleno ayuda a separar el contenido del borde.
Contenido / Content
Esta es el área donde se muestra el contenido del elemento, como texto, imágenes, etc.

- Anidado

Puedes ver la compatibilidad de los navegadores con esta característica en Can i use?

El anidado en CSS, utilizando el operador &, fue introducido como una característica en preprocesadores de CSS como Sass (Syntactically Awesome Stylesheets) y Less (Leaner CSS). Esta característica permite hacer referencia al selector padre dentro de un conjunto de reglas CSS anidadas. Aunque no inicio como una característica nativa de CSS puro, se ha vuelto muy popular debido a su capacidad para mejorar la legibilidad y mantenibilidad del código CSS. El anidado permite estructurar el código CSS de manera más intuitiva, especialmente cuando se trabaja con estilos para componentes que tienen una estructura jerárquica. Permite escribir estilos de forma más concisa al evitar la repetición de selectores comunes. El anidado ya forma parte de las caracteristicas nativas de CSS, puedes escribir reglas CSS anidadas como esta:

Ejemplo

Antes (sin anidado)
					
	<!-- index.html -->
	<!DOCTYPE html>
	<html lang="es">
	<head>
		<meta charset="UTF-8">
		<meta name="viewport" content="width=device-width, initial-scale=1.0">
		<link rel="stylesheet" href="styles.css">
		<title>Document</title>
	</head>
	<body>
		<div class="card">
			<h2 class="card-title">Título de la tarjeta</h2>
			<p class="card-text">Texto de la tarjeta</p>
		</div>
	</body>
	</html>
					
				
					
	/* styles.css */
	.card {
		border: 1px solid #000;
		padding: 10px;
	}
	.card-title {
		font-size: 20px;
	}
	.card-text {
		font-size: 16px;
	}
Ahora (con anidado '&' )
					
	/* styles.css */
	.card {
		border: 1px solid #000;
		padding: 10px;
		
		&.card-title {
			font-size: 20px;
		}
		&.card-text {
			font-size: 16px;
		}
	}

Bootstrap

- ¿Qué es Bootstrap?

Es una libreria front-end que se utiliza para hacer el diseño y diseños responsivos. Esta basada en un sistema de clases que se aplican a los elementos HTML para darles estilos y comportamientos predefinidos.

- ¿Como instalarla?

Para instalar Bootstrap en tu proyecto, puedes hacerlo de dos maneras:

CDN
Puedes agregar Bootstrap a tu página web fácilmente utilizando el CDN (Content Delivery Network) de Bootstrap.
							
	<!DOCTYPE html>
	<html lang="en">
	<head>
		<meta charset="UTF-8">
		<meta name="viewport" content="width=device-width, initial-scale=1.0">
	
		<!-- Bootstrap CSS -->
		<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.2.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-rb65" crossorigin="anonymous">
	
		<title>Bootstrap</title>
	</head>
	<body>
		
		<!-- Bootstrap JS -->
		<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.2.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-keI4" crossorigin="anonymous"></script>
	</body>
	</html>
Descargar
Para descargar Bootstrap la forma mas sencilla es usando el link que se proporciona como CDN y darle a guardar como. Esto nos permitira guardar sus 2 archivos como bootstrap.min.css y bootstrap.bundle.min.js. Y ahora los podemos agregar a nuestro proyecto como se muestra en el ejemplo anterior. 
							
	<!DOCTYPE html>
	<html lang="en">
	<head>
		<meta charset="UTF-8">
		<meta name="viewport" content="width=device-width, initial-scale=1.0">
	
		<!-- Bootstrap CSS descargado -->
		<link href="./bootstrap.min.css" rel="stylesheet"/>
	
		<title>Bootstrap</title>
	</head>
	<body>
		
		<!-- Bootstrap JS descargado -->
		<script src="./bootstrap.bundle.min.js"></script>
	</body>
	</html>

  • El CSS de Bootstrap se importa en el head.
  • El JS de Bootstrap se importa al final del body

- Contenedores

Bootstrap utiliza un sistema de contenedores para ayudar a organizar y alinear el contenido de una página web. Los contenedores son elementos HTML que envuelven el contenido de la página y ayudan a controlar su ancho y alineación. Hay tres tipos de contenedores en Bootstrap:

.container
El cual define un ancho máximo a cada punto de quiebre.
.container-fluid
El cual define el ancho máximo a cada punto de quiebre.
.container-{breakpoint}
Define el ancho al 100% asta el punto de quiebre especifico.
