HTML

¿Qué es HTML?

Que va a ir y en donde va ir

HTML, o HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar utilizado para crear y diseñar páginas web. Es un lenguaje de marcado que permite estructurar el contenido de una página web mediante el uso de etiquetas o elementos que describen la semántica y la estructura del documento.
En términos más simples, HTML proporciona la estructura básica de una página web, definiendo los diferentes elementos que componen la página, como encabezados, párrafos, listas, enlaces, imágenes, formularios, entre otros. Estos elementos se organizan mediante etiquetas HTML, que son instrucciones delimitadas por corchetes angulares
("<" y ">") que indican al navegador web cómo mostrar el contenido. HTML es fundamental para la creación de páginas web ya que proporciona la base sobre la cual se construye el contenido y la apariencia visual de un sitio web.
Trabaja en conjunto con otros lenguajes como CSS (Cascading Style Sheets) para el diseño y la presentación visual, y JavaScript para la interactividad y la funcionalidad dinámica. Juntos, estos tres lenguajes forman la tríada básica de desarrollo web.

Estructura básica de HTML

	<!-- Aqui le decimos al explorador el lenguaje en el que esta escrito la página -->

- Imágenes

Representadas por la etiqueta <img>, se utilizan para incrustar imágenes en una página web. Se debe proporcionar la URL de la imagen en el atributo "src".

Código Resultado
<img src="..."/>
Atributo Descripción
src Este atributo es usado para definir la ruta de la imágen que usaremos, puede ser una ruta de la web o una ruta relativa
alt Este atributo es usado para definir un texto alterativo por si la imágen no llega cargar o ser encontrada.
weight Este atributo es usado para definir la anchura de la imagen a utlizar, por defecto usa pixeles como medida.
height Este atributo es usado para definir la altura de la imagen a utlizar, por defecto usa pixeles como medida.

CSS

¿Qué es CSS?

Como se ven las cosas.

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para definir la presentación y el aspecto visual de un documento HTML. Con CSS, puedes controlar la apariencia de elementos HTML, como el color, el tamaño, la fuente y el diseño. CSS permite separar la estructura del contenido HTML de su estilo, lo que facilita la creación de sitios web con un diseño coherente y flexible. Además, al utilizar reglas de estilo, CSS permite aplicar estilos de manera eficiente y consistente en todo un sitio web.

- Propiedades y valores

Las propiedades son características que deseas cambiar, como el color o el tamaño del texto. Los valores son los ajustes específicos que le das a esas propiedades. 

						
	p {
		/* Cambia el color del texto a azul */
		color: blue; 
		
		/* Tamaño de la letra en 16 píxeles */
		font-size: 16px;
	}

- Selectores

Por elemento
								
	p {
		color: red;
	}
Por clase
							
	.clase {
		color: red;
	}
Por ID
							
	#id {
		color: red;
	}
Selecciona todo (*)
							
	* {
		/* Las reglas que escribamos con este selector 
		se aplicaran a todos los elementos, 
		si es que estos no tiene reglas ya definidas */
	}
Combinación de clases
							
	.clase1.clase2 {
		/* Estilos para elementos que tienen 
		ambas clases "clase1" y "clase2" */
	}
Elemento con clase
							
	p.clase {
		/* Estilos para elementos p que también tienen 
		la clase "clase" */
	}
Selectores multiples
							
	p, .clase, #id {
		/* Estilos aplicados a los elementos p, .clase y #id */
	}

Este es párrafo color café

- El modelo de caja

El modelo de caja es un concepto fundamental en CSS que describe cómo se representan los elementos HTML en una página web. Cada elemento HTML se representa como una caja rectangular que consta de cuatro áreas principales:

Margen / Margin
Es el espacio entre el borde de la caja y los elementos adyacentes. Los márgenes ayudan a controlar el espacio entre los diferentes elementos en la página.
Borde / Border
Es la línea que rodea el contenido y el relleno de la caja. Puede tener un grosor, un estilo (como sólido, punteado, etc.) y un color definidos.
Relleno / Padding
Es el espacio entre el contenido y el borde de la caja. El relleno ayuda a separar el contenido del borde.
Contenido / Content
Esta es el área donde se muestra el contenido del elemento, como texto, imágenes, etc.

- Anidado

Puedes ver la compatibilidad de los navegadores con esta característica en Can i use?

El anidado en CSS, utilizando el operador &, fue introducido como una característica en preprocesadores de CSS como Sass (Syntactically Awesome Stylesheets) y Less (Leaner CSS). Esta característica permite hacer referencia al selector padre dentro de un conjunto de reglas CSS anidadas. Aunque no inicio como una característica nativa de CSS puro, se ha vuelto muy popular debido a su capacidad para mejorar la legibilidad y mantenibilidad del código CSS. El anidado permite estructurar el código CSS de manera más intuitiva, especialmente cuando se trabaja con estilos para componentes que tienen una estructura jerárquica. Permite escribir estilos de forma más concisa al evitar la repetición de selectores comunes. El anidado ya forma parte de las caracteristicas nativas de CSS, puedes escribir reglas CSS anidadas como esta:

Ejemplo

Antes (sin anidado)
					
	<!-- index.html -->
	<!DOCTYPE html>
	<html lang="es">
	<head>
		<meta charset="UTF-8">
		<meta name="viewport" content="width=device-width, initial-scale=1.0">
		<link rel="stylesheet" href="styles.css">
		<title>Document</title>
	</head>
	<body>
		<div class="card">
			<h2 class="card-title">Título de la tarjeta</h2>
			<p class="card-text">Texto de la tarjeta</p>
		</div>
	</body>
	</html>
					
				
					
	/* styles.css */
	.card {
		border: 1px solid #000;
		padding: 10px;
	}
	.card-title {
		font-size: 20px;
	}
	.card-text {
		font-size: 16px;
	}
Ahora (con anidado '&' )
					
	/* styles.css */
	.card {
		border: 1px solid #000;
		padding: 10px;
		
		&.card-title {
			font-size: 20px;
		}
		&.card-text {
			font-size: 16px;
		}
	}

Bootstrap

- ¿Qué es Bootstrap?

Es una libreria front-end que se utiliza para hacer el diseño y diseños responsivos. Esta basada en un sistema de clases que se aplican a los elementos HTML para darles estilos y comportamientos predefinidos.

- ¿Como instalarla?

Para instalar Bootstrap en tu proyecto, puedes hacerlo de dos maneras:

CDN
Puedes agregar Bootstrap a tu página web fácilmente utilizando el CDN (Content Delivery Network) de Bootstrap.
							
	<!DOCTYPE html>
	<html lang="en">
	<head>
		<meta charset="UTF-8">
		<meta name="viewport" content="width=device-width, initial-scale=1.0">
	
		<!-- Bootstrap CSS -->
		<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.2.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-rb65" crossorigin="anonymous">
	
		<title>Bootstrap</title>
	</head>
	<body>
		
		<!-- Bootstrap JS -->
		<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.2.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-keI4" crossorigin="anonymous"></script>
	</body>
	</html>
Descargar
Para descargar Bootstrap la forma mas sencilla es usando el link que se proporciona como CDN y darle a guardar como. Esto nos permitira guardar sus 2 archivos como bootstrap.min.css y bootstrap.bundle.min.js. Y ahora los podemos agregar a nuestro proyecto como se muestra en el ejemplo anterior. 
							
	<!DOCTYPE html>
	<html lang="en">
	<head>
		<meta charset="UTF-8">
		<meta name="viewport" content="width=device-width, initial-scale=1.0">
	
		<!-- Bootstrap CSS descargado -->
		<link href="./bootstrap.min.css" rel="stylesheet"/>
	
		<title>Bootstrap</title>
	</head>
	<body>
		
		<!-- Bootstrap JS descargado -->
		<script src="./bootstrap.bundle.min.js"></script>
	</body>
	</html>

  • El CSS de Bootstrap se importa en el head.
  • El JS de Bootstrap se importa al final del body

- Contenedores

.container

.container-fluid
El cual define el ancho máximo a cada punto de quiebre.
.container-{breakpoint}
Define el ancho al 100% asta el punto de quiebre especifico.

JavaScript

¿Qué es JavaScript/JS?

JS es un lenguaje interpretado de alto nivel.

JavaScript es un lenguaje de programación popular que se utiliza principalmente para crear interactividad en páginas web. Es un lenguaje de programación de alto nivel, interpretado por los navegadores web para mejorar la funcionalidad y el diseño de los sitios web. Con JavaScript, los desarrolladores pueden crear efectos visuales dinámicos, validar formularios, manipular contenido, gestionar eventos del usuario y mucho más. Además de su uso en el desarrollo web, JavaScript también se utiliza en otros entornos, como el desarrollo de aplicaciones móviles y de escritorio.

- ¿Como usar JavaScript?

Existen tres formas de usar JavaScript en un documento HTML:

En línea
								
	<button onclick="alert('Hola mundo!')">Haz click</button>
Incrustado
								
	<script>
		alert('Hola mundo!');
	</script>
Externo
								
	<!-- index.html -->
	<!DOCTYPE html>
	<html lang="es">
	<head>
		<meta charset="UTF-8">
		<meta name="viewport" content="width=device-width, initial-scale=1.0">
		<title>Ejemplo de JavaScript</title>

	<body>
		<button onclick="saludar()">Haz click</button>

		<!-- Aqui enlazamos el archivo de JavaScript -->
		<script src="app.js"></script>
	</body>
	</html>

- Condicionales

Los condicionales en JavaScript se utilizan para tomar decisiones basadas en ciertas condiciones. Algunos de los condicionales más comunes en JavaScript son:

If
							
	// Si la condición es verdadera, se ejecuta el bloque de código
	if (condición) {
		// Bloque de código
	}
Else
							
	// Si la condición es falsa, se ejecuta el bloque de código
	else {
		// Bloque de código
	}
Else if
							
	// Si la condición anterior es falsa, se evalúa esta condición
	else if (condición) {
		// Bloque de código
	}
Switch
							
	// Evalúa una expresión y ejecuta el bloque de código correspondiente
	switch (expresión) {
		case valor1:
			// Bloque de código
			break;

			// Bloque de código

		default:
			// Bloque de código
	}

Los bucles en JavaScript se utilizan para ejecutar un bloque de código varias veces. Algunos de los bucles más comunes en JavaScript son:

For

For of

Los eventos en JavaScript se utilizan para detectar acciones del usuario, como hacer clic en un botón, desplazarse por una página web, escribir en un campo de texto, etc. Algunos de los eventos más comunes en JavaScript son:

- DOM

El DOM (Document Object Model) es una interfaz de programación para documentos HTML y XML. El DOM representa la estructura de un documento como un árbol de nodos, donde cada nodo representa una parte del documento (por ejemplo, un elemento HTML, un atributo, un comentario, etc.). El DOM es una interfaz de programación que permite a los programadores manipular la estructura, el estilo y el contenido de un documento.

El DOM se utiliza en JavaScript para acceder y manipular elementos HTML. Algunos de los métodos más comunes para acceder y manipular el DOM en JavaScript son:

getElementById
							
	// Obtiene un elemento por su ID
	document.getElementById('id');
getElementsByClassName
							
	// Obtiene elementos por su clase
	document.getElementsByClassName('clase');
getElementsByTagName
							
	// Obtiene elementos por su etiqueta
	document.getElementsByTagName('etiqueta');
querySelector
							
	// Obtiene el primer elemento que coincida con un selector
	document.querySelector('selector');
querySelectorAll
							
	// Obtiene todos los elementos que coincidan con un selector
	document.querySelectorAll('selector');

- AJAX

El AJAX (Asynchronous JavaScript And XML) es una técnica de programación que se utiliza para crear aplicaciones web interactivas. Permite a las aplicaciones web enviar y recibir datos del servidor de forma asíncrona, sin interferir con el comportamiento de la página web. El AJAX se utiliza en JavaScript para enviar y recibir datos del servidor sin tener que recargar la página.

Algunos de los métodos más comunes para enviar y recibir datos del servidor utilizando AJAX en JavaScript son:

XMLHttpRequest
							
	// Crea una nueva instancia de XMLHttpRequest
	let xhr = new XMLHttpRequest();
open
							
	// Abre una nueva solicitud
	xhr.open('GET', 'archivo.txt', true);
send
							
	// Envía la solicitud al servidor
	xhr.send();
onreadystatechange
							
	// Define una función que se ejecuta cuando cambia el estado de la solicitud
	xhr.onreadystatechange = function() {
		// Bloque de código
	};

- JSON

El JSON (JavaScript Object Notation) es un formato de intercambio de datos ligero y fácil de leer. Se utiliza para enviar y recibir datos entre un servidor y un cliente. El JSON se utiliza en JavaScript para almacenar y transmitir datos de forma estructurada.

Algunos de los métodos más comunes para trabajar con JSON en JavaScript son:

parse
							
	// Convierte una cadena JSON en un objeto JavaScript
	JSON.parse('{"nombre": "Juan", "edad": 25}');
stringify
							
	// Convierte un objeto JavaScript en una cadena JSON
	JSON.stringify({nombre: 'Juan', edad: 25});

- ECMAScript

El ECMAScript es una especificación de lenguaje de programación que se utiliza para definir el estándar de JavaScript. La especificación de ECMAScript define la sintaxis, los tipos de datos, los objetos y los métodos que se pueden utilizar en JavaScript. La especificación de ECMAScript se actualiza periódicamente para incluir nuevas características y funcionalidades en JavaScript.

Algunas de las características más recientes de ECMAScript incluyen:

  • Arrow functions
  • Template literals
  • Let y const

- Promesas

Las promesas en JavaScript se utilizan para manejar operaciones asíncronas. Una promesa es un objeto que representa el resultado de una operación asíncrona, y puede estar en uno de los siguientes estados:

  • Pendiente
  • Resuelta
  • Rechazada

- Async y await

El async y await en JavaScript se utilizan para manejar operaciones asíncronas de forma más sencilla. El async y await son palabras clave que se utilizan para definir funciones asíncronas y esperar el resultado de una operación asíncrona.

Algunos de los métodos más comunes para trabajar con async y await en JavaScript son:

async
							
	// Define una función asíncrona
	async function nombreFuncion() {
		// Bloque de código
	}
await
							
	// Espera el resultado de una operación asíncrona
	let resultado = await operaciónAsíncrona();

- Módulos

Los módulos en JavaScript se utilizan para organizar y reutilizar código. Los módulos son archivos que contienen código JavaScript y se utilizan para exportar e importar funciones, variables y clases entre diferentes archivos.

Algunos de los métodos más comunes para trabajar con módulos en JavaScript son:

Export
							
	// Exporta una función, variable o clase
	export function nombreFuncion() {
		// Bloque de código
	}
Import
							
	// Importa una función, variable o clase
	import {nombreFuncion} from './archivo.js';

- Errores

Los errores en JavaScript se utilizan para manejar situaciones inesperadas que pueden ocurrir durante la ejecución de un programa. Algunos de los errores más comunes en JavaScript son:

SyntaxError
							
	// Error de sintaxis
	throw new SyntaxError('Mensaje de error');
ReferenceError
							
	// Error de referencia
	throw new ReferenceError('Mensaje de error');
TypeError
							
	// Error de tipo
	throw new TypeError('Mensaje de error');
RangeError
							
	// Error de rango
	throw new RangeError('Mensaje de error');

jQuery

- ¿Qué es jQuery?

Encontrar un elemento y hacer algo con el.
jQuery permite a los desarrolladores escribir menos código para lograr los mismos efectos que se lograrían con JavaScript puro, lo que resulta en un desarrollo más rápido y eficiente. Ofrece una amplia gama de funciones para seleccionar elementos HTML, manipular su contenido, modificar estilos, realizar animaciones y trabajar con AJAX para cargar datos de forma asíncrona.

- ¿Como usar jQuery?

Para usar jQuery en un documento HTML, primero debemos incluir la biblioteca de jQuery en el encabezado del documento. Luego, podemos escribir código jQuery entre las etiquetas <script> en el cuerpo del documento o en un archivo externo con la extensión .js. 

					
	<!-- Incluimos la biblioteca de jQuery -->
	<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

Una vez cargado el src podemos crear nuestro archivo app.js y empezar a escribir nuestro código jQuery.

- Ready event

Ofrece una manera de ejecutar codigo JS tan pronto como el DOM de nuestra pagina sea seguro de manipular. Basicamanete espera a que todo el HTML termine de cargar para poder ejecutar codigo JS. 

					
	// Esto es una buena practica
	$(document).ready(function(){
		// Bloque de código
	})

- Atributos y clases

Algunos pueden ser utilizados como Getters (obtener el dato) y otros pueden ser utilizados como Setters (establecer el dato)

attr()
Es para controlar atributos de la etiqueta.
html()
Para modificar el HTML.
text()
Se utiliza para obtener o establecer el texto contenido dentro de los elementos seleccionados.
val()
Es para modificar o extraer la información dada por el usuario.
					
		// Como Getter
		var texto = $('#id').text()

		// Como Setter
		$('#id').text('Nuevo texto')

- Ejemplo de metodos

Mostraremos una serie de métodos de jQuery que se utilizan para manipular atributos y clases de elementos HTML seleccionados.

$('#id').attr('atributNombre', 'valor');
Este método se utiliza para establecer el valor de un atributo específico en el elemento seleccionado.
$('#id').removeAttr('atributNombre');
Establece el atributo atributoNombre del elemento con el id "id" al valor "valor".
$('#id').addClass('claseNombre');
Agrega una clase específica al elemento seleccionado
$('#id').removeClass('claseNombre');
Elimina una clase específica del elemento seleccionado.
$('#id').toggleClass('claseNombre');
Alterna la presencia de una clase específica en el elemento seleccionado. Si el elemento ya tiene la clase, la eliminará; si no la tiene, la agregará. Es útil para alternar estados como activo/inactivo o visible/oculto.

- Traversing

Es el hecho de navegar por el DOM

Familia:

.parent()
Selecciona el elemento padre directo del elemento seleccionado.
.parents()
Selecciona todos los ancestros del elemento seleccionado.
.siblings()
Selecciona todos los elementos hermanos del elemento seleccionado.
.children()
Selecciona todos los elementos secundarios directos del elemento seleccionado.

Proximidad:

.closest()
Selecciona el elemento más cercano que coincide con un selector, empezando por el propio elemento y ascendiendo en el DOM hasta alcanzar el documento.
.next()
Selecciona el siguiente elemento inmediato del elemento seleccionado.
.prev()
Selecciona el elemento anterior inmediato del elemento seleccionado.
.nextAll()
Selecciona todos los elementos siguientes del elemento seleccionado.
.prevAll()
Selecciona todos los elementos anteriores del elemento seleccionado.

- Manejo de eventos

					
	/* Esta línea establece un evento personalizado llamado 'handler' 
	en el elemento con el id "id". Cuando se activa este evento, se ejecuta la función proporcionada 
	como segundo argumento. Este método proporciona una forma de vincular múltiples eventos al 
	mismo elemento y manejarlos con una sola función. */
	$('#id').on('handler', function(){/* Tu codigo */});

	/* Esta línea activa manualmente el evento 'click' en el elemento con el id "id". 
	Esto simula el comportamiento de hacer clic en el elemento. */
	$('#id').trigger('click');
	
	/*  Esta línea vincula la función proporcionada como argumento para ejecutarse 
	cuando se hace clic en el elemento con el id "id". Es una forma abreviada de asignar 
	un controlador de eventos al evento 'click'. */
	$('#id').click(function(){/* Tu codigo */});

	/* Esta línea vincula la función proporcionada como argumento para ejecutarse 
	cuando el puntero del mouse se desplaza sobre el elemento con el id "id". La función se ejecuta tanto 
	cuando el puntero del mouse entra como cuando sale del elemento. */
	$('#id').hover(function(){/* Tu codigo */});

	/* Esta línea vincula la función proporcionada como argumento para ejecutarse 
	cuando el elemento con el id "id" recibe el foco. Por ejemplo, cuando un campo de entrada de texto 
	es seleccionado por el usuario. */
	$('#id').focus(function(){/* Tu codigo */});

	/* Esta línea vincula la función proporcionada como argumento para ejecutarse 
	cuando se realiza un evento de alternancia en el elemento con el id "id". Sin embargo, hay un error en 
	esta línea. El método .toggle() en jQuery se utiliza para alternar la visibilidad de un elemento, 
	no para manejar eventos. Parece que se ha cometido un error y debería revisarse. */
	$('#id').toogle(function(){/* Tu codigo */});
